💥 Similarity Search System using Deep Embeddings💥 

Developed a fast and accurate similarity search pipeline using deep learning-based embeddings.
➡️ Fine tuned a pre-trained model to extract 512-dimensional embeddings from face/image/text data
 ➡️ Applied cosine similarity to compare query embeddings with a database of stored vectors
 ➡️ Implemented scalable search using Faiss and Scikit-learn, achieving 92.4% top-5 accuracy and <100ms query latency on a 10K+ embedding set
 ➡️ Normalized embeddings to optimize accuracy and performance
 ➡️ Visualized embedding clusters using t-SNE and UMAP — query samples (circled in red) landed near their respective class clusters, validating embedding quality
📌 Use Case: Face recognition, duplicate detection, recommendation system
🛠️ Tech stack: Python, PyTorch/TensorFlow, Faiss, Scikit-learn, Matplotlib
